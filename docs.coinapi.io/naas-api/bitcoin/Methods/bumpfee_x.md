bumpfee
=======

Overview[​](/naas-api/bitcoin/Methods/bumpfee_x#overview "Direct link to Overview")
-----------------------------------------------------------------------------------

The **'bumpfee'** Bumps the fee of an opt-in-RBF transaction T, replacing it with a new transaction B.
