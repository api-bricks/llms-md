===============================

Overview[​](/naas-api/ethereum/Methods/eth_getUncleCountByBlockNumber#overview "Direct link to Overview")
---------------------------------------------------------------------------------------------------------

The **'eth\_getUncleCountByBlockNumber'** method returns the number of uncles in a block from a block matching the given block number.
