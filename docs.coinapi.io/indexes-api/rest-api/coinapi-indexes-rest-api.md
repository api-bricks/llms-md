========================

Authentication[​](/indexes-api/rest-api/coinapi-indexes-rest-api#authentication "Direct link to Authentication")
----------------------------------------------------------------------------------------------------------------

* HTTP: Bearer Auth

| --- | --- |
| Security Scheme Type: | apiKey |
| Header parameter name: | Authorization |

| HTTP Authorization Scheme: | bearer |
